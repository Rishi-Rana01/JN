# 🏛️ JG University - Premium Institutional Interface

A sophisticated, editorial-style landing page for **JG University**, meticulously crafted to embody excellence in education and innovation. This project represents a transition from a tech-centric aesthetic to a timeless, premium institutional design.

---

## 🤝 Human-AI Collaboration
This project is a product of a **60/40 Human-AI partnership**:
- **Human (60%)**: Core logic, project initialization, initial component structure, content strategy, and creative direction.
- **AI (40%)**: Refactoring the design system, implementing advanced animations, optimizing styling for high-end aesthetics (Serif conversion), and ensuring cross-browser responsiveness.

### How AI Enhanced This Project
The AI didn't just "write code"; it served as a design and performance consultant:
*   **Design Transformation**: Re-engineered the visual identity from a generic dark-mode interface to a refined **Ivory & Burnished Gold** Serif system.
*   **Micro-Animations**: Leveraged **Framer Motion** to implement subtle, high-end motion patterns (scroll reveals, staggered entry animations) that give the site a "premium" feel.
*   **Architectural Polish**: Refactored ad-hoc styling into a centralized token-based system using **Tailwind CSS 4** variables.
*   **Institutional Imagery**: Optimized the integration of official university assets for better visual storytelling.

---

## ✨ Key Features
- **Editorial Serif Design**: A clean ivory background paired with sophisticated typography.
- **Responsive Layout**: Seamless experience across Mobile, Tablet, and Desktop.
- **Dynamic Components**: Interactive Hero section, Program cards, and Testimonials.
- **Modern Tech Stack**: Built with the latest versions of Next.js and React.
- **Performance Optimized**: Lazy-loaded components and optimized image delivery.

---

## 🛠️ Tech Stack
- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/jg-university.git
cd jg-university
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## ☁️ Deployment on Vercel

Deploying this project is straightforward. Follow these steps:

### Method 1: Vercel Dashboard (Recommended)
1.  **Push your code** to a GitHub/GitLab/Bitbucket repository.
2.  Go to [Vercel](https://vercel.com/new).
3.  **Import** your repository.
4.  Vercel will automatically detect **Next.js**.
5.  Click **Deploy**.

### Method 2: Vercel CLI
If you have the Vercel CLI installed:
```bash
npm i -g vercel
vercel
```

### Build Configuration
Ensure your project settings in Vercel match:
- **Framework Preset**: `Next.js`
- **Build Command**: `npm run build`
- **Output Directory**: `.next`
